# Интерактивная карта граффити Раевки

Этот проект — сайт с интерактивной картой граффити села Раевка.

Его цель — сохранить, задокументировать и систематизировать работы местных уличных художников. Особое внимание в проекте уделяется художнику с псевдонимом **LAZY**, чьи работы можно встретить в разных частях села.

Проект задумывался как своеобразный **цифровой архив местного стрит-арта**, чтобы работы не терялись со временем и были доступны для просмотра всем заинтересованным.
